# Bekannte Probleme
Folgende Probleme sind bekannt. An der Behebung wird gearbeitet.

## Debian - Anzeige "virtualisiert"
Debian wird durch /dev/vhost-vsock als "virtualisiert" angezeigt

## Debian - Lockscreen
Unter Debian (gnome/wayland) kann im abgesicherten Modus der Lockscreen nicht genutzt werden.

## WLAN-Info
In MacOS kann die WLAN-Information nicht abgerufen werden. Auch unter Windows findet dies statt, sofern die Standort-Berechtigung nicht erteilt wurde.
